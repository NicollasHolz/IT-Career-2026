# Cleaning Quote Calculator

A simple quote calculator for small service businesses.

## Features

- Add customer name
- Select service type
- Add estimated hours
- Add hourly rate
- Add extra fee
- Add discount
- Calculate total price
- Save quotes using localStorage
- Delete individual quotes
- Clear all saved quotes

## Technologies

- HTML
- JavaScript
- localStorage
- Git and GitHub

## Why I built this project

I built this project to practise JavaScript fundamentals and create a useful tool for small service businesses such as cleaning, car detailing, pressure washing and furniture delivery.
